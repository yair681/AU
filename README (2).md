# 🌟 מערכת הסכמי הגן הקסום

מערכת מלאה לניהול הסכמי הורים בגן ילדים.

---

## 📁 מבנה הקבצים

```
gan-system/
├── server/
│   └── index.js          ← השרת הראשי (Node.js + Express)
├── public/
│   └── index.html        ← טופס הורים (פרונטאנד)
├── admin/
│   └── index.html        ← פאנל ניהול
├── data/
│   ├── submissions.json  ← כל ההגשות (נוצר אוטומטית)
│   └── form-config.json  ← הגדרות הטופס (נוצר אוטומטית)
├── package.json
└── README.md
```

---

## 🚀 התקנה והפעלה

### שלב 1 — התקן Node.js
הורד מ: https://nodejs.org (גרסה 18 ומעלה)

### שלב 2 — התקן תלויות
פתח טרמינל בתיקיית `gan-system` והרץ:
```bash
npm install
```

### שלב 3 — הפעל את השרת
```bash
npm start
```

השרת יעלה על פורט 3000. תראה בטרמינל:
```
🌟 גן הקסום — Server running on http://localhost:3000
📋 Parent form: http://localhost:3000/
🔐 Admin panel: http://localhost:3000/admin/
🔑 Default password: hila2025
```

## ✨ תכונות המערכת

### טופס הורים:
- ✅ טעינה דינמית של תוכן הטופס מהשרת
- ✅ כל שדות ההסכם, הרשמה, בריאות, חופשות
- ✅ 5 שדות חתימה דיגיטלית (עכבר ומגע)
- ✅ ולידציה לפני שליחה
- ✅ שמירה מיידית בשרת

### פאנל ניהול:
- ✅ לוח בקרה עם סטטיסטיקות
- ✅ רשימת כל ההגשות עם חיפוש
- ✅ צפייה מלאה בכל פרטי הגשה כולל חתימות
- ✅ **הדפסה** — לחץ 🖨️ לכל הגשה
- ✅ מחיקת הגשות
- ✅ **עריכת כל תוכן הטופס** — טקסט הסכם, שעות, חופשות
- ✅ שינוי סיסמת מנהל

## ☁️ פרסום לאינטרנט (אופציונלי)

### אפשרות מומלצת — Railway:
1. צור חשבון ב https://railway.app
2. חבר את תיקיית הפרויקט
3. Railway יזהה אוטומטית Node.js

### אפשרות נוספת — Render:
1. צור חשבון ב https://render.com
2. "New Web Service" → חבר GitHub
3. Build: `npm install`
4. Start: `npm start`
